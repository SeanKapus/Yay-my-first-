Card Filters

Using a filter, get a random card to display from the list.

{{ cards|random }} {% with cards|random as random_card %} {{ random_card.rank }} - {{ random_card.get_suit_display }} {% endwith %}

Using a filter, display only the last card from the list.

{{ cards.all.reverse|first }} {% with cards.all.reverse|first as last_card %} {{ last_card.rank }} - {{ last_card.get_suit_display }} {% endwith %}
